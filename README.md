# Blue.tcc Macro | KUROMI

Blue.tcc Macro | KUROMI is an AutoHotkey v2-based macro utility for Sailor Piece / Roblox gameplay support, featuring a premium Kuromi-inspired GUI, preset management, auto clicker, melee swap, and tower reset automation.

> Original base concept and credit: **Yato**  
> Customized GUI / enhanced version: **Blue.tcc**

---

## Features

- 💜 Premium Kuromi-themed GUI
- ⌨️ Up to 10 customizable key task slots
- 💾 Preset management (create / save / delete / load)
- ⚙️ Mode selection:
  - Normal
  - Boss Rush
  - Infinite Tower
- 🖱️ Auto Clicker with configurable delay
- 🥊 Melee Swap with saved screen positions
- 🏰 Tower Reset automation
- 🧪 Debug Log window
- 🎮 Custom toggle hotkey support (F1 / Mouse buttons / F-keys)
