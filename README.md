# Blue.tcc Macro | KUROMI 💜🖤

**Blue.tcc Macro** is an **AutoHotkey v2.0 script** designed to enhance gameplay automation on PC.  

### Features
- 🖱️ **Auto Clicker**: Click automatically with adjustable speed.  
- ⚔️ **Melee Swap**: Auto-switch melee weapons for Boss Rush / Infinite Tower modes.  
- 🏰 **Tower Reset**: Automatically reset tower runs.  
- 🎛️ **Custom GUI**: Manage presets, key binds, delays, and macro settings.  
- 💻 **Debug Log**: Monitor macro actions and troubleshoot issues through the GUI.  
- ⏱️ **Runtime Tracker**: Display the macro’s runtime.

---

## 🛠️ Requirements
- Windows 10 / 11  
- [AutoHotkey v2.0+](https://www.autohotkey.com/)  
- The game window must be **active** while the macro runs.

---

## ⚡ Installation
1. Download all files from this repository.  
2. Make sure **AutoHotkey v2** is installed.  
3. Double-click the `.ahk` file to run the macro.  
4. On first run, `AutoSub.ini` will be created automatically to store presets.

---

## 🎮 Usage
- Press **F1** (default toggle key) to enable/disable the macro.  
- Select a **preset** from the GUI or create a new one.  
- Configure **key binds** and **delays** for each slot.  
- Enable features according to the mode:  
  - **Normal**: Standard macro.  
  - **Boss Rush**: Activates Melee Swap.  
  - **Infinite Tower**: Activates Melee Swap + Tower Reset.  
- Save melee positions with **Set Melee 1 / 2** buttons.

---

## ⚙️ Configuration
- The `AutoSub.ini` file stores **presets**, **key binds**, **delays**, and **melee positions**.  
- Settings can be edited via the GUI or directly in the `.ini` file.

---

## 📝 Notes
- GUI automatically resizes and shows/hides features based on the selected mode.  
- Tower Reset runs every X minutes according to GUI settings.  
- Debug Log can be enabled/disabled for monitoring macro activity.
